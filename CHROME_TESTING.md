## 已知问题和解决方案

### 1. 如果页面监控列表不显示

**检查步骤**：
```javascript
// 在 Console 中运行
document.getElementById('crow-monitor-overlay')
```
- 如果返回 `null`：插件未正确注入
- 如果返回元素：检查 CSS display 属性

**解决方案**：
1. 刷新页面 (F5)
2. 重新加载插件
3. 检查控制台是否有错误

### 2. 如果提示音不播放

这在 Chrome 上应该能正常工作，但如果有问题：

**检查步骤**：
1. 确保浏览器允许声音
2. 检查音量设置
3. 查看控制台是否有 AudioContext 错误

### 3. 如果数据不同步

**解决方案**：
1. 打开 Chrome DevTools → Application → Storage
2. 检查 chrome.storage.local 中的数据
3. 点击插件的"清空"按钮重置

## 如果需要创建 Manifest V3 版本

Chrome 未来会逐步淘汰 Manifest V2（预计 2025 年），如果需要升级：

1. 将 `manifest_version: 2` 改为 `3`
2. 将 `browser_action` 改为 `action`
3. 将 `background.scripts` 改为 `background.service_worker`
4. 更新权限声明

目前 **不需要**升级，Manifest V2 在 Chrome 144 上完全可用。

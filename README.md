# 足球Crow*指数监控插件

这是一个浏览器扩展（Chrome / Microsoft Edge），用于实时监控 titan007.com 网站上的足球比赛 Crow* 指数变化。当指数变化超过设定阈值时，插件会自动捕获并记录相关数据。

## 功能特点

- 🔍 **实时监控**：自动监控页面上的Crow*指数变化
- 📊 **变化检测**：当指数变化超过设定阈值（默认10%）时自动捕获
- 📱 **弹出窗口**：在浏览器上方显示监控状态和捕获的数据
- 💾 **数据存储**：本地存储监控数据，支持导出功能
- ⚙️ **自定义设置**：可调整变化阈值和刷新间隔
- 🎯 **智能过滤**：自动清理已完成或无效的比赛数据

## 安装方法

### 1. 下载插件文件
将所有插件文件下载到本地文件夹

### 2. 加载到 Chrome / Edge 浏览器
本仓库默认的 manifest.json 为 Manifest V2，用于兼容 Microsoft Edge Chromium 84。

1. 打开Chrome浏览器，输入地址：`chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择包含插件文件的文件夹（manifest.json 所在目录）
5. 插件图标将出现在浏览器工具栏中

#### Microsoft Edge（Chromium）加载方式
1. 打开 Edge，输入地址：`edge://extensions/`
2. 开启左侧/右上角的“开发人员模式”
3. 点击“加载解压缩的扩展”
4. 选择包含插件文件的文件夹（manifest.json 所在目录）

#### 使用 Manifest V3（新版 Chrome / Edge）
仓库内同时提供了 `manifest.mv3.json`。如果你需要在支持 MV3 的新版浏览器中使用：
1. 将 `manifest.json` 重命名为 `manifest.mv2.json`
2. 将 `manifest.mv3.json` 重命名为 `manifest.json`
3. 重新在扩展管理页加载

## 使用方法

### 基本使用
1. 访问目标网站：https://live.titan007.com/indexall.aspx
2. 插件会自动开始监控页面上的Crow*指数
3. 点击浏览器工具栏上的插件图标查看监控状态

### 插件界面功能

#### 主界面
- **状态指示器**：显示当前监控状态（绿色表示活跃）
- **活跃比赛数**：当前正在监控的比赛数量
- **总变化数**：检测到的Crow*指数变化总数

#### 控制按钮
- **🔄 刷新**：手动刷新监控数据
- **🗑️ 清空**：清空所有监控数据
- **📊 导出**：将监控数据导出为JSON文件

#### 设置选项
- **变化阈值**：设置Crow*指数变化检测的百分比阈值（1-50%）
- **刷新间隔**：设置数据刷新频率（3-30秒）

### 数据展示
每个监控的比赛显示以下信息：
- 对阵双方球队名称
- 当前比分
- 比赛时间和状态
- Crow*指数变化详情（主胜和客胜）
- 变化幅度百分比
- 最后更新时间

### 工作原理
1. **内容脚本**（content.js）注入到目标页面，定期解析比赛数据
2. **变化检测**：比较当前数据与历史数据，计算Crow*指数变化百分比
3. **消息传递**：检测到显著变化时，通过Chrome消息API发送给后台脚本
4. **数据存储**：后台脚本（background.js）接收并存储变化数据
5. **界面展示**：弹出窗口（popup.html）显示监控状态和数据列表

### 变化检测算法
插件使用以下算法检测Crow*指数变化：

```javascript
const homeChange = Math.abs(current.crowIndex.home - previous.crowIndex.home);
const awayChange = Math.abs(current.crowIndex.away - previous.crowIndex.away);

const homeChangePercent = (homeChange / previous.crowIndex.home) * 100;
const awayChangePercent = (awayChange / previous.crowIndex.away) * 100;

// 如果任一指数变化超过设定阈值，则记录该变化
if (homeChangePercent > threshold || awayChangePercent > threshold) {
    // 记录显著变化
}
```

## 注意事项

1. **网站结构变化**：如果目标网站的HTML结构发生变化，可能需要更新选择器
2. **数据准确性**：插件依赖于页面上的数据，确保目标网站数据准确
3. **性能影响**：建议设置合理的刷新间隔，避免过于频繁的检查
4. **隐私保护**：所有数据仅存储在本地，不会上传到任何服务器

## 故障排除

### 插件无法正常工作
1. 检查是否正确加载了插件
2. 确认访问的是正确的目标网址
3. 查看浏览器控制台是否有错误信息
4. 尝试重新加载插件

### 数据不更新
1. 检查网络连接是否正常
2. 确认目标网站数据是否在更新
3. 尝试调整刷新间隔设置
4. 手动点击刷新按钮

### 图标不显示
1. 确认图标文件存在于插件目录中
2. 检查manifest.json中的图标路径是否正确
3. 尝试重新加载插件
